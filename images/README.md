# 画像フォルダ

5S活動の注意箇所のイラスト・写真をここに置きます。

## 命名規則

```
{エリアID}-{内容の説明}.jpg
```

例：
- `area-A1-shelf.jpg` … 現場A-1の「工具棚」
- `area-A1-passage.jpg` … 現場A-1の「通路」
- `area-B1-forklift.jpg` … 現場B-1の「フォークリフト」

- ファイル名は **半角英数字とハイフン** のみ（日本語・スペース不可）
- 拡張子は `.jpg`（写真）または `.png`（図・イラスト）

## サイズ・容量の目安

| 項目       | 推奨値                          |
| ---------- | ------------------------------- |
| 横幅       | 800〜1200px 程度                |
| 容量       | **1枚あたり 500KB 以下**        |
| 形式       | JPEG（写真）/ PNG（図）         |

> スマホで3秒以内に表示するため、容量は必ず守ってください。
> 容量が大きいときは、無料の画像圧縮ツール（例：[Squoosh](https://squoosh.app/)）で縮小できます。

## JSON への記載方法

`areas/area-XX.json` の項目に次のように書きます。

```json
"image": "images/area-A1-shelf.jpg",
"imageAlt": "工具棚の正しい整理例"
```

- `image` … このフォルダからの相対パス
- `imageAlt` … 画像が表示できないときや読み上げ用の説明（日本語）

画像が無い項目は `"image": ""` にするか、その行を削除してください。
